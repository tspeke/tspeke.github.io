    Audio and Videos

  • <audio> : Inserts sound content into HTML document. Supports .mp3, .wav and .ogg
    • src : Directs to location of audio file
    • controls : Adds ability to play/pause and mute audio
    • Note that some browsers support different file types and may have different default settings
    • loop : Automatically restarts audio once finished
    • muted : Boolean attribute which sets audio default to be muted
  • <source> : Source elements can be nested within media element. This causes the browser to select the first source that it understands
  • <video> : Inserts video content into HTML document. Supports .mp4, .webm and .ogg
    • All attributes introduced in <audio> element are supported in <video>
    • autoplay : Boolean attribute which causes video to play on loading
    • width : Sets the width of the video screen
    • poster : Displays an image with the destination given when video is downloading
