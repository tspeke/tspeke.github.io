Website Recipe

Learn how to format a website using HTML code

Ingredients

  • "DOCTYPE" declaration
  • "html" element containing all code
  • "header" containing elements:
    • "title"
    • "meta"
  • "body" containing elements:
    • "p"
    • "img"
    • "h..."
    • "ul", "ol" and "li"
    • "a"
  • "footer" containing elements:
    • "a"
    • "p"

Instructions

  1. Establish core code
  2. Link styling elements in the header section
  3. Write the content of the webpage in the body section
  4. Incorporate links, pictures and figures where appropriate
