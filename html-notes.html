Introduction

The following are some basic HTML formatting operators:

  • <h1> to <h6> : Different levels of heading, where <h1> is largest
  • <p> : paragraphs of text
  • <br> : Introduces a forced line break - does not require a closing parameter

  • <b> : Bold text
  • <strong> : Important
  • <i> : Italic text
  • <em> : Emphasis
  • <mark> : Highlight
  • Bulleted list:
    • <ul>
    <li>...</li>
    <li>...</li>
</ul>
  • Numbered list:
    • <ol>
    <li>...</li>
    <li>...</li>
</ol>
  • Tiered list:
    • <ul>
    <li>..1..</li>
    <li>..2..
        <ul>
            <li>..a..</li>

    • Target URL given by href
  • <img src="image.jpg" alt="Description of image"> : Image element

    • alt giving text if image cannot load
  • <pre> : Preformatted block preserving space and line breaks typed within
  • <code> : Inline code snippet
  • <blockquote> :
    Blockquote

Lesson 1: Basics of HTML

Attributes

An "attribute" is a value placed inside the opening tag of an HTML element. Attributes provide additional information about the element or specify how the element should behave. The standard syntax for an attribute is:

<element attribute="value"></element>

Common attributes and their purposes include:

  • href : Used within <a> to specify destination URL of a link
  • target : Used within <a> to specify where to open link E.g:
    • target="_blank" enables the link to open in a new tab
  • src : Used within <img> to specify the source of an image
  • alt : Used within <img> to provide alternative descriptive text for the image
