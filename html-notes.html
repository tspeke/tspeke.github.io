Introduction

The following are some basic HTML formatting operators:

  • <h1> to <h6> : Different levels of heading, where <h1> is largest
  • <p> : paragraphs of text
  • <br> : Introduces a forced line break - does not require a closing parameter

  • <b> : Bold text
  • <strong> : Important
  • <i> : Italic text
  • <em> : Emphasis
  • <mark> : Highlight
  • Bulleted list:
    • <ul>
    <li>...</li>
    <li>...</li>
</ul>
  • Numbered list:
    • <ol>
    <li>...</li>
    <li>...</li>
</ol>
  • Tiered list:
    • <ul>
    <li>..1..</li>
    <li>..2..
        <ul>
            <li>..a..</li>
            <li>..b..</li>
        </ul>
    </li>
    <li>..3..</li>
</ul>
  • <a href="https://example.com"> : Hyperlink
    • Target URL given by href
  • <img src="image.jpg" alt="Description of image"> : Image element
    • src giving location of image relative to root
    • alt giving text if image cannot load
  • <pre> : Preformatted block preserving space and line breaks typed within
  • <code> : Inline code snippet
  • <blockquote> :
    Blockquote
  • <hr> : Horizontal line for separating sections
  • <!-- ... --> : Comment
  • When recording code the strings & l t ; and & g t ; (written without spaces) will output < and > characters

Lesson 1: Basics of HTML

Attributes

An "attribute" is a value placed inside the opening tag of an HTML element. Attributes provide additional information about the element or specify how the element should behave.

<element attribute="value"></element>
